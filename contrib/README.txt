This is the folder for any contributor's original DMP, FUR, etc. source mods

DO NOT PLACE FILES DIRECTLY IN THIS FOLDER
PLACE THEM IN THE APPROPRIATE TRACKER SUB-FOLDER

Acceptable formats for this branch of the tree:

.fur, .dmp, .vgm, .ZSM, and accompanying .inf / .txt files.
(one format per song please)

If submitting in VGM or ZSM format, please place it into the folder
appropriate to the tracker used to create the work.

Thanks for your contributions!
